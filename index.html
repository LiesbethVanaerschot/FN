Welcome!

On this little static site you can find the styling of the one pager: Nucleair Forum for Fukushima. This site is generated with the very fast tool Hugo which is a static site generator.

On this site you can find three subpages: basics, blocks and structures. What do they mean?

  • Basics

    Here you can find all the basic styling for a site like colors, fonts, backgrounds,... These basics can be used for styling the whole website.

  • Blocks

    Blocks are the first level elements for building the content of your website. Here you can find styled bodycopy, links, buttons, lists,...

  • Structures

    Structures are large pieces of a website. They consist most of the time out of blocks tweaked with some available basics.
